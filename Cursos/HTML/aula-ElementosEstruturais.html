ELEMENTOS ESTRUTURAIS

no html temos alguns elementos estruturais que ajudam a organizar o conteudo

ela é usada para separar conteudos no site por blocos

a div é uma caixa para organizar a estrutura do site em setores ou blocos


Meu teste de span

o span é usado para separar conteudos dentro de um texto, ele não cria uma nova linha

SEÇÃO

a seção é usada para separar conteudos por temas ou assuntos

ela é usada para separar conteudos que tenham o mesmo tema ou assunto

ARTIGO

o artigo é usado para separar conteudos que podem ser reproduzidos em outros sites

ela é usada para separar conteudos que podem ser reproduzidos em outros sites, como noticias, posts de blogs, etc
