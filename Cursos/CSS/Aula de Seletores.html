MINHA PRIMEIRA AULA DE CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML e XML. Ele permite separar o conteúdo da estrutura do documento, facilitando a manutenção e o design das páginas web.

Principais Conceitos do CSS

  • Seletores: Permitem selecionar elementos HTML para aplicar estilos específicos.
  • Propriedades: Definem os aspectos visuais dos elementos, como cor, fonte, margem, etc.
  • Valores: Especificam os valores das propriedades, como "red" para cor ou "16px" para tamanho de fonte.
  • Classes e IDs: Permitem aplicar estilos a grupos específicos de elementos ou a um único elemento.
  • Box Model: Conceito que descreve a estrutura de um elemento HTML, incluindo conteúdo, padding, border e margin.
  • Media Queries: Permitem criar designs responsivos que se adaptam a diferentes tamanhos de tela.
