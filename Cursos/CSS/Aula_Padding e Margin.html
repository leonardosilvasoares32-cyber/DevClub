Box Model

O modelo de caixa (Box Model) é um conceito fundamental no CSS que descreve a estrutura e o layout dos elementos HTML na página. Cada elemento é representado como uma caixa retangular composta por quatro áreas principais: conteúdo, preenchimento (padding), borda (border) e margem (margin).

1. Conteúdo (Content): Esta é a área onde o conteúdo real do elemento é exibido, como texto, imagens ou outros elementos HTML.

2. Preenchimento (Padding): O preenchimento é o espaço entre o conteúdo do elemento e sua borda. Ele cria um espaço interno dentro da caixa, afastando o conteúdo da borda.

3. Borda (Border): A borda é a linha que envolve o conteúdo e o preenchimento do elemento. Ela pode ter diferentes estilos, larguras e cores.

4. Margem (Margin): A margem é o espaço externo ao redor da borda do elemento. Ela cria um espaço entre o elemento e outros elementos na página, afastando-os uns dos outros.

O Box Model é essencial para entender como os elementos são posicionados e espaçados na página, permitindo aos desenvolvedores web controlar o layout e a aparência dos elementos de forma eficaz.
